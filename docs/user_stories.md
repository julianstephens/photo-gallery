## Users

- As a user, I can view all guilds that I have access to.
- As a user, I can filter the dashboard by guild.
- As a user, I can set a default guild to automatically filter by.
- As a user, I can view all photo galleries for the filtered guild.
- As a user, I can select a photo gallery to view for the filtered guild.
- As a user, I can select an image to preview and download.

## Admin Users

- As an admin user, I can create a new photo gallery in the filtered guild.
- As an admin user, I can edit the name of a specific photo gallery in the filtered guild.
- As an admin user, I can upload photos to a specific photo gallery in the filtered guild.
- As an admin user, I can monitor upload(s) for a specific photo gallery in the filtered guild.
- As an admin user, I can remove a single photo from a specific photo gallery in the filtered guild.
- As an admin user, I can select and remove multiple photos from a specific photo gallery in the filtered guild.
- As an admin user, I can delete a specific photo gallery in the filtered guild.
- As an admin user, I can view my requests.
- As an admin user, I can create a request.
- As an admin user, I can cancel my open request.
- As an admin user, I can leave a comment on my open requests.

## Super Admin Users

- As a super admin user, I can view all requests.
- As a super admin user, I can approve, deny, or cancel open requests.
- As a super admin user, I can re-open closed requests.
- As a super admin user, I can leave a comment on a request, regardless of its status.
- As a super admin user, I can delete a request.
